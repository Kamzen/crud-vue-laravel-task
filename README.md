### Project setup


### To Run frontend

```
cd frontend-vue
```

```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```

### To Run Backend

```
cd api-laravel
```

```
composer install
```

```
npm run migrate
```

```
npm run seed
```

```
php artisan serve
```
